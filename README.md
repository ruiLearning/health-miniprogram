# 🌱 健康打卡小程序

## 功能介绍

| 页面 | 功能 |
|------|------|
| 🍽️ 饮食记录 | 按早/午/晚/加餐记录，35种食物库，热量+三大营养素自动计算 |
| ✅ 每日打卡 | 6项健康任务打卡，饮水记录，心情记录，体重记录 |
| 📊 数据统计 | 7天热量趋势图、营养分布图、体重变化图、BMI计算 |
| 👤 我的目标 | 身体信息录入，BMR/TDEE自动计算，热量目标设置 |

## 部署步骤

### 1. 准备工作
- 注册微信小程序账号：https://mp.weixin.qq.com
- 下载微信开发者工具：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 2. 导入项目
1. 打开微信开发者工具
2. 点击「+」新建项目
3. 选择「导入项目」
4. 目录选择本文件夹（`health-miniprogram`）
5. 填入你的 AppID（在微信公众平台获取）
6. 点击「导入」

### 3. 预览测试
- 点击工具栏「预览」按钮，用微信扫码在手机上查看
- 或点击「真机调试」进行调试

### 4. 上线发布
1. 工具栏点击「上传」
2. 填写版本号和项目备注
3. 登录微信公众平台 → 版本管理 → 提交审核
4. 审核通过后点击「发布」即可上线

### 5. 分享给他人
- 上线后用户可通过微信搜索小程序名称找到
- 或通过小程序码分享

```
├── app.js          # 全局逻辑 + 数据存储

├── sitemap.json    # SEO配置
├── images/         # TabBar图标
└── pages/
    ├── diet/       # 饮食记录页
    │   ├── diet.wxml
    │   ├── diet.wxss
    │   ├── diet.js
    │   └── diet.json
    ├── checkin/    # 每日打卡页
    ├── stats/      # 数据统计页
    └── profile/    # 我的目标页
```

## 数据存储

所有数据存储在用户手机本地（wx.setStorageSync），不需要服务器，保护隐私。

存储 key 说明：
- `hc_profile` — 用户基本信息和目标
- `hc_day_YYYY-MM-DD` — 每天的饮食/打卡/饮水数据

## 技术栈

- 微信小程序原生开发（WXML + WXSS + JS）
- 无任何第三方依赖
- Canvas 2D 绘制图表
- 本地存储（无需云服务）

## 注意事项

- 小程序 AppID 需要自行申请（免费）
- 个人主体可发布，无需企业认证
- 食物热量数据基于常见食物参考值，仅供参考
